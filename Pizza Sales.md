# Pizza Sales Analysis  
**Tools Used:** Excel • Power BI • Python (Jupyter Notebook)

## 1. Project Overview
This project analyzes a Pizza Sales dataset using Excel, Power BI, and Python.  
The objective is to understand sales patterns, identify top-performing pizzas, analyze revenue trends, and generate business insights that can help support data-driven decision making.

## 3. Steps Performed

### **Excel**
- Cleaned dataset  
- Created pivot tables  
- Analyzed category-wise sales and order distribution  

### **Power BI Dashboard**
Includes:
- Total Sales  
- Total Orders  
- Revenue by Category  
- Top 5 & Bottom 5 Pizzas  
- Sales by Day and Hour  
- Interactive slicers (Category, Pizza Name, Size)

### **Python (Jupyter Notebook)**
- Loaded and cleaned dataset  
- Performed exploratory data analysis (EDA)  
- Validated insights obtained from Excel and Power BI  
- Created visual charts for sales patterns and category performance  

---

## 4. Key Insights
- Evening hours recorded the highest order volume  
- Classic pizzas contributed the highest revenue  
- Certain pizzas showed consistently low demand  
- Weekends performed better compared to weekdays
